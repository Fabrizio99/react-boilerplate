# React Boilerplate

A React.js boilerplate using Vite as a build tool.
This boilerplate uses Typescript as a programming language and Tailwind CSS for styling.
Testing is configured using Vitest and Testing Library

## Packages
- React Router DOM
- React Hook Form
- Vitest
- Classname
- Redux Toolkit
- Tailwind CSS

## Tools
- Eslint
- Prettier
